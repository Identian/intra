# Top-Delta

El proyecto esta generado en [Angular CLI](https://github.com/angular/angular-cli) version 15.0.0

## Desarrollo

Al ejecutar ng serve, la aplicación se desplegará en el puerto 4200 de localhost, tal como suele ocurrir con los proyectos de Angular.

## Build

En nuestro proyecto, contamos con diferentes entornos que se encuentran en la carpeta environments, donde encontrarás los distintos archivos de configuración. Para construir la aplicación, sigue estos pasos:

en qa `npm run qa` 

en producción `npm run production`

para entender como se compila puede ver la configuración en el archivo package.json y el detalle de la configuración en el angular.json en la llave build y luego en la llave configuration.
